# Proteomics: Peptide Identification from Mass Spectrometry

This project explores how experimental mass spectrometry data can be translated into a peptide sequence.  
In other words, it addresses the question:

**"Which peptide produced this mass spectrum?"**

## Purpose

The purpose of this project is to implement a bioinformatics algorithm that reconstructs a peptide from its mass spectrum. This is a fundamental problem in proteomics, where proteins are broken into peptides and analyzed using mass spectrometry.

The project specifically focuses on:

- Understanding how peptide fragmentation generates a mass spectrum  
- Generating theoretical spectra from candidate peptides  
- Comparing theoretical and experimental spectra  
- Identifying the best-matching peptide using a scoring approach
